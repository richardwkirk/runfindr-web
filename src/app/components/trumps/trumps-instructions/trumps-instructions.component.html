Instructions

Welcome to Runner Trumps! You can use this page to create Top Trump style cards for athletes using their parkrun history.

Adding your Trump cards

  • Select athletes using the menu above to add their trump cards (select magnifier -> athletes).
  • Select the 'x' on a card to remove it from the view.
  • Click the image on the card to add a picture. We support the following types of image:
    • Public images from the internet
      1. Find a public URL for the image you want to import.
      2. Paste the URL into the input box.
    • A facebook profile image found by entering a facebook profile id. To get a facebook profile id:
      1. Visit https://lookup-id.com/ to get the profile id.
      2. You need to copy in a facebook profile URL such as: https://www.facebook.com/zuck
      3. Copy the id found into the pop-up you see after selecting the image.

Printing your Trump cards
